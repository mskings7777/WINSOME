# Winsome MVP

Winsome is a UI-first Expo prototype based on `docs/prd.md`.

## Run

```sh
npm install
npm run start
```

## MVP Scope

- WhatsApp-like genre and personality navigation
- Local mock data only
- Add Genre and Add Personality modals
- Collectible, gamified personality cards
- Static chat preview with dummy messages and a non-functional input
- No backend, auth, AI chat, memory, notifications, or database
