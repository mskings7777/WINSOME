# Design System: Vivid Modernity

## Brand & Style
The brand identity is "Vivid Modernity"—expressive, disciplined, and energetic. The style is a blend of **Corporate Modern** efficiency and **High-Contrast** visual energy. By moving from sharp edges to rounded corners and adopting a punchy crimson-led palette, the UI evokes approachability and innovation.

## Colors
The color system drives action through high-fidelity vibrance.
- **Primary:** #e61b5a (Crimson Action)
- **Secondary:** #c84c64 (Muted Rose)
- **Tertiary:** #008d50 (Success Green)
- **Neutral:** #8a7173 (Warm Gray-Pink)
- **Mode:** Light

## Typography
The system uses **Inter** for all text to ensure a contemporary digital feel.
- **Headlines:** Bold weights (600-700) for strong hierarchy.
- **Body:** Regular weight with a 1.5x line-height ratio for readability.
- **Labels:** Medium weight with 0.5px letter spacing for functional UI elements.

## Layout & Spacing
A fluid 8px grid system ensures consistency across all screen sizes.
- **Base Unit:** 8px
- **Gutter:** 16px
- **Margins:** 24px
- **Internal Padding:** 16px (Standard)

## Elevation & Depth
Hierarchy is conveyed through **Ambient Shadows**. Surfaces use soft, diffused, low-opacity shadows tinted with the neutral color (#8a7173). This creates a sense of soft layers and clean depth without adding visual clutter.

## Shapes
The system uses a **Rounded** shape language:
- **Base Radius:** 8px (0.5rem) for buttons and inputs.
- **Large Radius:** 16px (1rem) for cards and containers.
- This softens the visual impact of the high-contrast color palette.

## Components
- **Buttons:** 8px rounded corners, primary crimson fill for high-priority actions.
- **Inputs:** 8px rounded corners, neutral borders, crimson focus states.
- **Cards:** 16px rounded corners with soft ambient shadows for content grouping.
- **Chips:** Pill-shaped (fully rounded) using tertiary and secondary colors for status and categories.
